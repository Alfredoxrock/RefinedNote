Refined Notes — Installer exit code reference

File: Refined-Notes-1.0.0-x64.exe

Published: 2025-10-17

Overview

This page lists the exit/return codes emitted by the Refined Notes Win32 installer and how to interpret them.

General rules

  • 0 — Success. The installer completed successfully.
  • Non-zero — Failure or special condition. See mapping below for meaning and remediation.

Installer return code mapping

Code: 0
Meaning: Success — Installation completed.
Action: None required.
Code: 1
Meaning: General error — an unexpected error occurred during installation.
Likely causes: Disk I/O error, corrupt package, or unhandled internal error.
Action: Check installer logs at %LOCALAPPDATA%\RefinedNotes\installer.log (or the log path provided by the installer), collect verbose logs, and retry installation.
Code: 2
Meaning: User canceled.
Likely causes: User clicked Cancel or closed the installer UI.
Action: No action required; user-initiated cancellation.
Code: 3
Meaning: Insufficient privileges.
Likely causes: Installer requires elevation but was run without Administrator rights.
Action: Re-run installer with Administrator privileges (right-click → Run as administrator).
Code: 4
Meaning: Upgrade/patch failed.
Likely causes: Version conflict or previous install in inconsistent state.
Action: Uninstall previous version and retry, or check installer logs for specific failure details.
Code: 5
Meaning: Network download failure (if installer downloads additional payload).
Likely causes: Network outage, proxy blocking, certificate validation failed.
Action: Check network connectivity and proxy settings; retry.
Code: 1001
Meaning: Failed to acquire lock on application data directory.
Likely causes: Another instance running or folder locked.
Action: Close other instances and retry; if persistent, remove the lockfile at %LOCALAPPDATA%\RefinedNotes\lockfile and retry.
Code: 2001
Meaning: Invalid installer signature or certificate chain not trusted.
Likely causes: Package not signed with a CA-trusted certificate.
Action: Use the signed package from our distribution URL or contact support and provide the installer SHA256.
